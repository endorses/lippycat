# Glossary

> **This appendix is planned for a future release.** It will define key terms used throughout the manual.

| Term | Definition |
|------|-----------|
| **BPF** | Berkeley Packet Filter — kernel-level packet filtering |
| **Hunter** | Edge capture node that forwards packets to a processor |
| **PCAP** | Packet Capture — standard file format for storing network packets |
| **Processor** | Central aggregation node that receives and analyzes packets from hunters |
| **RTP** | Real-time Transport Protocol — carries VoIP media streams |
| **SIP** | Session Initiation Protocol — VoIP signaling protocol |
| **Tap** | Standalone node combining hunter and processor capabilities |
| **TUI** | Terminal User Interface — interactive terminal-based monitoring |
